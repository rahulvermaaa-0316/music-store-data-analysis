# 🎵 Music Store Data Analysis (SQL Project)

## 📌 Project Overview
This project analyzes a digital music store database using SQL to extract business insights about customers, sales, and music trends.

## 🗄 Database Tables
The dataset includes the following tables:

- Artist
- Album
- Track
- Genre
- MediaType
- Customer
- Employee
- Invoice
- InvoiceLine
- Playlist
- PlaylistTrack

## 📊 Key Business Questions

1. Who is the senior most employee?
2. Which country has the highest number of invoices?
3. What are the top 3 invoice totals?
4. Which city generates the highest revenue?
5. Who is the best customer?

## 🔍 Advanced Analysis

- Rock music listeners identification
- Top rock artists based on number of tracks
- Songs longer than average length
- Most popular genre by country
- Top spending customers by country
